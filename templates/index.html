📖 API Reference

GET/api/status
Returns cached inverter readings. Polled every ~10s. Never hits the inverter directly.
curl http:///api/status
Response includes: solar_w, house_w, battery_w, grid_w, battery_percent, max_charge_power_w, max_discharge_power_w, energy_today_kwh, and more.
POST/api/control
Send commands to the inverter. All power values are in watts.
# Set to normal mode (restore saved charge + discharge rates) curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "normal"}' # Pause (zero charge + discharge rate) curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "pause"}' # Force charge at full rate (schedule 00:00-23:59, unlimited) curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "charge"}' # Force discharge at full rate (schedule 00:00-23:59, unlimited) curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "discharge"}' # Set max charge rate in watts curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "set_charge_power", "watts": 1500}' # Set max discharge rate in watts curl -X POST http:///api/control \ -H 'Content-Type: application/json' \ -d '{"action": "set_discharge_power", "watts": 1500}'
POST/api/config
Set the inverter IP address. Persisted across restarts.
curl -X POST http:///api/config \ -H 'Content-Type: application/json' \ -d '{"ip": "10.0.1.17"}'
GET/api/log
Returns the last 200 API request log entries as JSON.
curl http:///api/log
GET/api/polling
Check whether background polling is currently enabled.
curl http:///api/polling
POST/api/polling
Start or stop background polling of the inverter.
# Stop polling curl -X POST http:///api/polling \ -H 'Content-Type: application/json' \ -d '{"enabled": false}' # Start polling curl -X POST http:///api/polling \ -H 'Content-Type: application/json' \ -d '{"enabled": true}'
